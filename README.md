# Codo

**云原生个人生活工作助理 Agent。一次部署，处处使用。**

你只管收藏，Codo 把信息消费的时间成本归零。

刷到好文章没时间读、公众号文章堆积、群消息刷不完、好内容找不回来——这些原本需要你主动花时间的信息消费动作，Codo 全部替你完成。云端全天候运行，你不在电脑旁边，它依然在工作。

---

## 为什么是云原生

云原生不只是"部署在云上"，而是一套设计哲学：

- **容器化**：每个服务打包成 Docker 镜像，本地和云端行为完全一致
- **无状态**：服务本身不存数据，随时可重启、扩容，不影响用户
- **自动部署**：推代码自动触发构建和部署，零停机滚动更新
- **弹性伸缩**：任务量大时自动扩容 Worker，任务少时缩回，按需使用

结果：**你不在电脑旁边，Codo 依然在云端替你工作。**

---

## 核心交互模型

一切在后台自动发生。你只需要做一件事：**收藏或订阅**。

```
收藏一个链接         → 自动抓取总结，沉淀知识库
星标一个公众号       → 每篇新文章自动获取并提炼精华
订阅一个 RSS / 网站  → 新内容自动汇总，每日推送
```

久而久之，知识库成为只属于你的第二大脑，可随时检索和引用。

## 设计原则

- 人驱动，系统自动执行，结果永久沉淀
- 一次部署，手机 / 电脑 / 微信全端可用
- 数据自己掌控，私有部署
- 先解决自己的问题，再开源
